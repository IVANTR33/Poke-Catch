Installation:  
termux/cmd/powerShell

1. write  pkg install git
2. write  pkg install nodejs
3. write  git clone https://github.com/IVANTR33/Poke-Catch.git
4. write  cd ~/ Poke-Catch
5. write  npm install
6. write  node index.js
6 The program will ask you to enter the account token.
7. Once the token is validated, the account will ask you to enter the ID of the account that will be responsible for executing commands.
 And that would be it, you'd be online :)



 Steps 1 and 2 are exclusive to Termux; search for information on YouTube about how to install Node.js on your PC.
